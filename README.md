# sg_waste_management
Code for data pre-processing for a dashboard on sg's waste management 

# Dataset used
[https://www.kaggle.com/datasets/jocelyndumlao/global-fashion-brands](https://www.kaggle.com/datasets/kingabzpro/singapore-waste-management/data?select=2003_2017_waste.csv) 
Source: Kaggle 
Author: Abid Ali Awan

# About the Project 
This project involves looking at Singapore's waste management system from 2003 to 2017 on a dashboard and looking at different waste types and identify any trends in the waste management system in Singapore 

# How to run the project 
1. Download the dataset from Kaggle:

2. Open the notebook in Google Colab:
   - Upload `SG Waste Management.ipynb` to Colab  

3. Upload the downloaded dataset into Colab 

4. Run all cells from top to bottom.
